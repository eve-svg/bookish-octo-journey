# bookish-octo-journey
Este é um projeto bem simples mas que pode ajudar muito quem não sabe.
Feito no P5, este projeto lhe ensina a mandar aparecer uma palavra quando o mouse passar na tela do projeto.
Além disso, tem comandos para fazer mudar a cor de fundo da tela, deixar em degradê na cor que você quiser.
É perfeito para quem quer começar a aprender a fazer algo e não sabe por onde começar.
